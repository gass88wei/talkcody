## 打开终端

TalkCody 内置了终端功能，您可以通过以下方式打开：

### UI 按钮

点击顶部工具栏右侧的终端图标按钮。

![TalkCody Terminal](https://cdn.talkcody.com/images/terminal.png)

### 快捷键

| 平台 | 快捷键 |
|------|--------|
| macOS | `Cmd + J` |
| Windows/Linux | `Ctrl + J` |

## 多标签页

终端支持多标签页，方便同时运行多个任务：

| 操作 | macOS | Windows/Linux |
|------|-------|---------------|
| 新建标签页 | `Cmd + T` | `Ctrl + T` |
| 下一个标签页 | `Cmd + →` | `Ctrl + →` |
| 上一个标签页 | `Cmd + ←` | `Ctrl + ←` |

您也可以点击终端标签栏的 **+** 按钮新建标签页，点击标签页的 **×** 按钮关闭标签页。

## 全屏终端

如果需要更大的终端空间，可以使用全屏功能：

点击终端面板右上角的 **全屏按钮**（方框图标）即可进入全屏模式。

再次点击或按相同按钮可退出全屏。
