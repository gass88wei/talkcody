## 什么是 Plan Mode?

Plan Mode（计划模式）是一种让 AI 在执行任务前先制定详细计划的工作模式。

在 Plan Mode 下，AI 会：

1. **收集信息** - 使用只读工具分析代码和需求
2. **制定计划** - 创建详细的实现方案
3. **等待批准** - 将计划呈现给您审查
4. **执行任务** - 只有在您批准后才开始修改文件

## 何时使用 Plan Mode?

Plan Mode 适合以下场景：

- **复杂任务** - 涉及多个文件或多步骤的修改
- **重要变更** - 需要在执行前审查方案的情况
- **学习目的** - 想了解 AI 如何分析和解决问题

## 如何开启 Plan Mode

在聊天输入框下方的工具栏中，找到 **Plan Mode** 开关：

- **开启状态**：AI 会先制定计划等待您批准
- **关闭状态**：AI 会直接执行任务（Act Mode）

![TalkCody Plan Mode](https://cdn.talkcody.com/images/plan-mode.png)

## 审批计划

当 AI 完成计划后，您会看到一个计划审批卡片，提供三个选项：

| 操作 | 说明 |
|------|------|
| **Approve** | 批准计划，AI 开始执行 |
| **Edit** | 编辑计划内容后再批准 |
| **Reject** | 拒绝计划，可以提供改进建议 |

批准计划后，Plan Mode 会自动关闭，AI 开始执行任务。
