## 如何打开新窗口

### UI 操作

1. 进入项目列表页面
2. 在目标项目卡片右上角点击 **更多菜单**（三个点图标）
3. 选择 **Open in New Window**

![TalkCody multi Window](https://cdn.talkcody.com/images/multi-windows.png)

### 快捷键

| 平台 | 快捷键 |
|------|--------|
| macOS | `Cmd + Shift + N` |
| Windows/Linux | `Ctrl + Shift + N` |

## 使用场景

多窗口功能在以下场景中特别有用：

### 多项目同时编辑

在不同窗口中打开不同项目，无需频繁切换，各窗口独立运行。

### 跨项目代码对比

左右并排两个窗口，方便对比不同项目的代码实现。

### 多显示器工作

将不同项目窗口分配到不同显示器，充分利用屏幕空间。
