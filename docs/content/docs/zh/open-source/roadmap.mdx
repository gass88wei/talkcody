## TalkCody 愿景

构建一套面向开发者的 全流程 AI Coding Agent 平台：
从写代码、测试、文档，到发布、上线、运营，实现端到端自动化开发，让编程变得更快、更便宜、更简单。

## TalkCody 里程碑1：Top AI Coding Agent

<Callout type="success">
目标： 最低成本+最快速度+最高正确率 生成可运行代码
</Callout>

### 1.1 速度

#### 1.1.1 多 Task 并行

一个 Project内，一个Task开发功能，一个Task 修复 bug，一个Task写文档，多个Task并行工作。

- [ ] 支持多个 conversation/task 并行执行
- [ ] 和 Git Worktree 集成
- [ ] 支持 task 后台执行
- [ ] 支持 task 队列
- [ ] Task 完整的生命周期管理
- [ ] UI 交互更新

#### 1.1.2 多 Agent 协作并行

实现 Orchestrator-Worker 模式：主 Agent 拆解任务，Coder、Tester、Writer Agent 并行执行。

#### 1.1.3 支持 1到2小时的 long run tasks

### 1.2 成本

- [ ] Compaction 策略优化
- [ ] 支持 list file methods tool
- [ ] 支持 TOON encoding/decoding
- [ ] 优化每个tool的输出 token
- [ ] web fetch tool 加入 AI 分析和总结

### 1.3 正确性

- [ ] 支持 Hook 机制
- [ ] 在 SWE-bench Verified 和 Terminal-Bench 上持续提升排名
- [ ] 权限和安全完善加强

### 1.4 易用性

- [ ] 和 Editor 更好的集成
- [ ] 和 Terminal 更好的集成
- [ ] 和 git 和 github 更好的集成
- [ ] 支持 AI 代码自动补全
- [ ] 支持模型的自动选择和路由
- [ ] 增强可观测性
